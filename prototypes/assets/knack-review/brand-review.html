Tensions surfaced

T-scale · observed

Type scale is ad-hoc (48 → 44 → 40 → 32 → 22 → 20px, ratios 1.09 / 1.10 / 1.25 / 1.45 / 1.10 — no consistent ratio). The page-builder/WordPress fingerprint. A redesign can impose a single modular scale for a more intentional typographic rhythm.

T-cta-vocab · observed

CTA voice is fragmented: three different sign-up labels (Start Building For Free, Get Started Free, Start Free Trial) and three see-more variants (Explore this solution, See More Solutions, See All Integrations). 21 distinct CTA labels overall. A canonical primary action would sharpen conversion.

T-radius-vocab · observed

Radius vocabulary is mixed (11 / 12 / 15 / 16px plus 9999px pill). 11px dominates but the small-radius spread reads as inconsistent. A redesign should pick one small radius.

T-tokens-unused · observed

The site ships a full Bootstrap + WordPress token layer (--bs-primary blue, --bs-* semantics, --wp--preset--*) whose framework defaults don't match the magenta brand — the real brand colors live in separate tokens (--pink, --dark-purple, --body-black). The token layer exists in name only; a redesign can collapse to a single coherent brand-token system.

T-logo-variants · observed

Only one logo variant was captured (inline SVG wordmark, magenta). The redesign will likely need monochrome / inverted variants for dark sections and the footer.
