03Deterministic tensions5 · mechanical · ships in brand-review
type-scaleType scale is ad-hoc

Captured heading sizes cluster at ratios 1.09–1.10 so everything shouts at one volume.

Fix: impose a single modular scale (perfect-fourth ~1.25); tighten hero tracking.
cta-vocabCTA voice is fragmented

21 distinct CTA labels: three sign-up phrasings and three see-more variants. No canonical action.

Fix: one canonical primary (hot-pink) repeated site-wide + a single quiet secondary text-link.
radius-vocabRadius vocabulary is mixed

Five small radii in use: 11px ×18, 12px ×7, 15px ×7, 16px ×6, plus a 9999px pill.

Fix: pick one small radius (11px); reserve the pill for the feature tabs only.
tokens-unusedThe token layer exists in name only

Dual token systems and Lato residue; components don't consume a single source of truth.

Fix: collapse to one coherent brand-token system and wire components to consume it.
logo-variantsOnly one logo variant exists

A single inline-SVG magenta wordmark was captured — no monochrome or inverted variant for dark sections or the footer.

Fix: generate a mono / inverted / SVG set; use the inverted mark on the new dark-magenta bands.
04Design & experience5 · design-director judgment
The hero is a 2021 no-code template

Centered/split layout, a superlative one-liner, two stacked CTAs against a small product visual. It tells rather than shows.

Fix: rebuild the hero to show the product building itself — the live-systems register renders records cascading in and a workflow firing.
Two primary actions fight in the hero

A hot-pink “Start Building For Free” and a magenta “Get Started Free” sit side by side, with a third sign-up phrasing elsewhere.

Fix: one canonical hot-pink primary, one quiet “Talk to sales” secondary text-link.
Weak hierarchy, weak depth

The flat type scale combines with uniformly soft shadows (rgba(26,24,29,0.1) 0 4px 4px) so hero and section heads read at one volume.

Fix: a two-step shadow ladder plus the modular scale's dominant hero size restore depth and rhythm.
The sharpest argument looks generic

The one ownable idea — the head-to-head “How Knack Stands Apart” vs Lovable, Bolt, Airtable, Bubble — is rendered in the same flat cards as everything else.

Fix: give the comparison real differentiation (a high-contrast dark panel) so the strongest argument is the most distinct moment.
The brand under-shows itself

The best asset — an interactive feature switcher — is buried mid-page at small scale, and the signature gradient appears only on thin accents.

Fix: promote and enlarge the live switcher; foreground the gradient as a confident brand surface.
05Accessibility3 · rendered page
landmarksLandmark structure is incomplete

No semantic <nav> / <main> / <footer> regions on the page.

Fix: emit semantic landmarks per template, baked into the migrate render.
motionThe animated hero has no text equivalent

A motion-driven hero with no accessible name or static fallback.

Fix: give the hero an aria-label and a visible headline, with motion gated behind prefers-reduced-motion.
WCAG 1.4.4The viewport disables pinch-zoom

maximum-scale=1 blocks zooming — a failure that hits low-vision users hardest.

Fix: drop maximum-scale and any user-scalable=no. Zero design cost, real gain.

ogOpen Graph is broken for sharing

og:description renders as a literal newline, og:title is generic, and og:image is an .svg most platforms refuse. Shared links look broken.

Fix: a real description, the real title, a 1200×630 PNG/JPG. One fix lifts every shared link.
headingsHeading sizes are near-identical

H1 (48px) and H2 (44px) sit 4px apart — the levels read as flat.

Fix: the modular scale gives each level a distinct size with the H1 clearly dominant.
h1-intentThe H1 leads with a claim, not intent

A superlative assertion (“The most secure app and database builder…”) rather than an intent-matched, benefit-led headline.

Fix: lead with the job-to-be-done; move the superlative into proof (comparison + ROI).
metaTitle and meta need a polish

Title carries stray whitespace and a duplicated keyword.

Fix: trim and de-duplicate; use the ~155-char budget for a differentiated benefit.
chromeA dated promo is baked into the chrome

A time-bound strip above the nav: “Get 65% off Starter Plans for a limited time.”

Fix: move time-bound offers into a dismissible, swappable slot so the chrome stays evergreen.
ai-searchNot built to be cited by AI search

/llms.txt returns 404 and there is no Open Knowledge Format bundle; the persuasion structure is weakly citable.

Fix: add an llms.txt / OKF bundle; restructure the comparison and ROI proof as clean, citable blocks.
