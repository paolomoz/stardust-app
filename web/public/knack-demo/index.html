02 — What's worth resolving

Tensions identified

Five specific observations from the captured surface — the load-bearing input for the redesign. Each is a concrete fix, not a vague "make it better."

dated-pattern

Generic SaaS-template hero

A centered/split hero with a superlative one-liner and two stacked CTAs against a small product visual — it tells rather than shows, when Knack's whole value is building working software.

ia-clutter

Competing primary actions

A hot-pink "Start Building For Free" and a magenta "Get Started Free" sit side by side, and the page carries 21 distinct CTA labels (incl. a third sign-up phrasing). No single canonical action.

contrast-density

Flat hierarchy

The type scale is ad-hoc — 48/44/40px headings sit so close that hero, section heads and sub-heads shout at the same volume. Uniformly soft shadows add weak depth. One tonal plane.

cliché

By-the-numbers persuasion

Logo wall + generic 6-up feature grid + stat row. The genuinely sharp idea — the head-to-head competitor comparison — is rendered in the same flat cards, so the one differentiated argument doesn't look differentiated.

missed-opportunity

The best asset is buried — and the brand under-shows itself

An interactive feature switcher (Build a Database · Edit with AI · Automate Workflows) lets a visitor flip through the product actually working — but it lives mid-page at small scale. And the distinctive magenta→coral gradient appears only on thin accents while the canvas stays mostly white.

03 — Three directions

The variants

All three are brand-faithful: the captured magenta/coral palette, Inter type, the real Knack logo and the actual product screenshots. They differ by axis — A fixes the current shape, B bets on color, C bets on motion — so no two overlap.

BDesign-team motivator

What if we amplified Knack's magenta?

"The brand's underused capability, foregrounded."

Captured trait amplified: color-ladder re-weighting. In the captured site, near-black text and white dominate the canvas and magenta is barely on screen. This variant lets the brand's most ownable color own the page.

  • Full-bleed magenta→coral gradient hero, type reversed in white
  • Dark-magenta alternating bands instead of white-on-white cards
  • The comparison becomes a high-contrast dark panel
  • Same content & order as A — the bet is color + composition, static

"The brand's third dimension."

05 — Notes

What's real vs. placeholder

The hero and feature-tab visuals are the actual product screenshots captured from knack.com (database, AI, automate, pages, users, publish), and the logo is Knack's real SVG wordmark. Customer-logo names, the integration tiles, the ROI figures and the testimonial are rendered as representative placeholders — knack.com presents these via embeds/animation that automated extraction couldn't read cleanly, so they should be swapped for real assets before anything ships.

Validation: all three render with zero console errors; Mode A pinned. Variant C additionally passes motion validation — every animation neutralizes to the static reference under reduced-motion, and no headings animate.
